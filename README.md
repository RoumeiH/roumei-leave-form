# 柔美飯店 · 假單產生器

讀取「柔美班表」Google Sheet，依員工 + 日期自動帶入班別時間，一鍵產生可列印的 A4 假單（請假單／加班單／未刷卡證明單）。可把多張不同假單收進列印清單，每 2 張排成一張 A4，列印後給員工簽名。

---

## 功能

- **讀班表**：貼上柔美班表 Google Sheet 連結自動讀取（建議），或上傳同格式 Excel。
- **✨ 批次自動產出（新）**：讀入班表後，一鍵自動掃全月，依規則批次產出草稿清單：
  - 班表寫「**年**」 → 請假單、特休假、時間 = 該員工平常工時（自動推算）
  - 班表寫「**國加 HH-HH**」 → 加班單、勾「加班費」、時間 = 後面接的時段
  - 班表寫「**休加 HH-HH**」 → 加班單、勾「補休」、時間 = 後面接的時段（含跨夜）
  - 其他班別（休/例/國/一般上班/未打卡等）不自動產出，由主管手動處理
  - 掃描範圍可選：整月一次全掃（預設）／掃到今天為止／指定日期區間
  - 草稿清單可修改時間、刪除、一鍵全部加入列印清單
- **手動開單**（例外處理）：批次抓不到的（忘打卡、未刷卡、特殊假別）沿用原本手動流程，選人 → 選日 → 選類型。
- **自動抓人**：依姓名比對，自動抓出名單成員，並自動去掉姓名後的數字。
- **自動帶入**：讀表頭「月/日」對應日期，選好員工 + 日期後自動填入當天上下班時間。班別為時間（如 `10-21`、`9-18`、`19-04`、`1030/1930`、`13-22`、`20-08`）會拆成上下班時段；休假類（`休／例／國／年／套／三套／龍／寧夏／柔` 等）會自動帶入該員工「平常上班時段」（從當月班表推算最常出現的時段），方便填請假單，仍可手動修改。`+2H`、`+2.5H` 等加班註記與 `卡／國加／休加` 前綴會自動忽略。
- **可手改**：自動帶入後欄位可調整；產生後也能點單據黃底欄位改字。
- **三種假單**：請假單、加班單、未刷卡證明單。
- **A4 兩聯（列印清單）**：每張填好後「加入列印清單」，可重複加入不同員工、不同類別、不同日期。列印時自動每 2 張排成一張 A4（上下聯），超過 2 張自動分頁；清單可調整順序、刪除。列印時操作介面自動隱藏。

---

## 使用方式

1. 開啟網頁。
2. **步驟 1**：貼上柔美班表 Google Sheet 連結（含當月分頁的 `#gid=` 那一頁），按「讀取班表」。或改用上傳 Excel。
3. **步驟 2**：選員工 → 確認民國年 → 選日期（月份會自動帶入；可從班表預覽直接點日期）。
4. **步驟 3**：選假單類型，確認自動帶入內容，按「＋ 加入列印清單」。
5. 重複步驟 2–4 加入其他假單。
6. 按「列印 / 存PDF」，每 2 張排成一張 A4 印出。

---

## 重要：請用「部署後的網址」開啟

用 `file://` 直接雙擊開 index.html 時，瀏覽器的 CORS 限制可能擋掉對 Google Sheet 的讀取。**請務必部署後用 https 網址開啟**（GitHub Pages 或 Vercel 都可），Google Sheet 讀取才會正常。

---

## 部署到 GitHub Pages（最簡單，免額外註冊）

1. 把四個檔案（index.html、app.js、README.md、vercel.json）上傳到 GitHub repo。
2. repo → Settings → 左側 Pages → Source 選「Deploy from a branch」→ Branch 選 `main`、資料夾 `/ (root)` → Save。
3. 等 1–2 分鐘，頁面上方會出現網址（如 `https://你的帳號.github.io/leave-form/`），打開即可使用。

## 部署到 Vercel

1. repo 匯入 vercel.com → New Project。
2. Framework 選 **Other** → Deploy（已附 vercel.json，免設定）。

---

## Google Sheet 共用設定

1. Google Sheet 右上「共用」。
2. 一般存取權改為「知道連結的任何人」、角色「檢視者」。
3. 複製**當月分頁**的連結（網址含 `#gid=`）貼進工具。

---

## 調整名單

要增減假單對象、改名字或工時，編輯 `app.js` 最上方的 `ROSTER` 表即可：

- **一般人員**（在班表上）：`小名: { full:'全名' }`。工具用小名比對班表抓班別，假單印全名。
- **不在班表的人員**（手動、固定工時）：`小名: { full:'全名', fixed:{start:'08:30', end:'17:30'} }`。選到此人時直接帶入固定工時，日期手動選（需先填月份），不需要班表也能開單。

例如目前「賴俐均」不在班表，設為固定 08:30–17:30，選到她就自動帶入該時段。
